# Python Library Template

Welcome to the documentation for **python-library-template**.

This template is built for fast, reliable **agentic development** in Python. It includes:

- A clean project layout
- Strong defaults for formatting, linting, and testing
- Clear LLM instructions to keep automated contributions consistent

## Template checklist

Before publishing your own project, make sure you:

- Replace `python-library-template` and `python_library_template`
- Update `pyproject.toml` metadata and URLs
- Update this documentation and `README.md`
